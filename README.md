# Калькулятор
Для реализации используются только `+` и `-`, а не стандартные функции python.

### Базовые функции:
- `+`  сложение,
- `-` вычитание, 
- `/` деление,
- `*` умножение,
- `%` остаток от деления,
- `^x` возведение в степень,
- `x^1/2` извлечение квадратного корня,
- решение простых уравнений с одним неизвестным, например, `2 + x = 14`
